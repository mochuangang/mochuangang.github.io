1.虚拟机安装Linux系统

第1步：在虚拟机管理界⾯中单击“开启此虚拟机”按钮后数秒就看到 RHEL 7.4 系统安装界⾯，如图 1-所示。

描述 含义
Install Red Hat Enterprise Linux 7.4 安装Redhat Linux7.4系统
Test this media & install Red Hat Enterprise Linux 7 校验光盘完整性后再安装
Troubleshooting 启动救援模式

第2步：修改内核参数net.ifnames=0 biosdevname=0，使网卡名统一为eth0或者eth1的方式。
接下来选择第一项按Tab键，进入到如下的界面

在光标后面填上net.ifnames=0 biosdevname=0，如图所示

按回车，返回到安装界面

再按一次回车键进入安装
1.选择语言

2.选择时间地区上海

3.键盘默认

4.语言选项

5.系统安装的位置，默认是光盘的，以后要是有批量安装的可以更改

6.软件的安装，选择最小化的安装时最安全的，也可以选择带可视化界面的安装，不过一般都用不上可视化的界面，这里选择最小化安装，软件也是最小化安装

7.系统安装的位置选择手动配置，点击完成

选择标准分区

选择分区磁盘大小
添加第一个分区 /boot分区 1G

2.添加第二个分区 swap分区 1G

3.第三个分区全部给/分区

9.关闭kdump

10.网卡及主机名配置，前面要是添加修改了net.ifnames=0 biosdevname=0 ，下面的网卡名会变为eth0、eth1

添加ip地址

配置第二块网卡ip



11.关掉security policy

12.全部改完后，点击开始安按钮进行安装


添加密码

添加账户

13.直到出现如下界面，表示系统已经安装完成，重启即可
