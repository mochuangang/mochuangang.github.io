2.Hive常用数据类型

Hive数据类型 长度 例子
int 4byte有符号整数 20
bigint 8byte有符号整数 20
double 双精度浮点数 3.14159
float 单精度浮点数 3.14159
string 字符系列。可以指定字符集。可以使用单引号或者双引号。 ‘now is the time’ “for all good men”

2.2 创建表

内部表 vs 外部表
内部表/MANAGED_TABLE：创建语法create table xxx;
删除表之后：HDFS和MySQL的数据都被删除了
外部表：创建语法create EXTERNAL table xxx;
删除表之后：MySQL的数据都被删除了，但是HDFS的数据是还存在的
